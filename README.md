This is a repository storing all of my solutions to proof problems assigned for a directed group study. 
This directed group study was conducted at UC Merced (Spring 2026), focused on the theoretical background of 
Topological Data Analysis (TDA).
The study primarily introduces concepts in point-set topology (Topological spaces, Connectedness, compactness)
and algebraic topology (Homotopy, Path-homotopy).
Topological Data Analysis like the concepts we are learning, is very useful for examining structure in clouds of data, as well as removing noise from the data to look at the underlying trend.
